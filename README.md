## 📊 DAX Measures

| Measure Name | DAX Formula |
|-------------|------------|
| Total Sales | `SUMX(Sales_data, Sales_data[Units Sold] * Sales_data[Price Per Unit])` |
| Total Quantity | `SUM(Sales_data[Units Sold])` |
| Transactions | `COUNTROWS(Sales_data)` |
| Average Price | `AVERAGE(Sales_data[Price Per Unit])` |

---

## 📖 Measure Descriptions

### 💰 Total Sales
Calculates the total revenue generated by multiplying the units sold with the price per unit for each transaction.

```DAX
Total Sales =
SUMX(
    Sales_data,
    Sales_data[Units Sold] * Sales_data[Price Per Unit]
)
```

### 📦 Total Quantity
Calculates the total number of units sold across all transactions.

```DAX
Total Quantity =
SUM(Sales_data[Units Sold])
```

### 🧾 Transactions
Counts the total number of sales transactions recorded in the dataset.

```DAX
Transactions =
COUNTROWS(Sales_data)
```

### 📈 Average Price
Calculates the average selling price per unit.

```DAX
Average Price =
AVERAGE(Sales_data[Price Per Unit])
```
