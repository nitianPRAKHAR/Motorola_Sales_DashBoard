## 📊 DAX Measures

| Measure Name | DAX Formula |
|-------------|------------|
| Total Sales | `SUMX(Sales_data, Sales_data[Units Sold] * Sales_data[Price Per Unit])` |
| Total Quantity | `SUM(Sales_data[Units Sold])` |
| Transactions | `COUNTROWS(Sales_data)` |
| Average Price | `AVERAGE(Sales_data[Price Per Unit])` |

---

## 📖 Measure Descriptions

### 💰 Total Sales
Calculates the total revenue generated by multiplying the units sold with the price per unit for each transaction.

```DAX
Total Sales =
SUMX(
    Sales_data,
    Sales_data[Units Sold] * Sales_data[Price Per Unit]
)
```

### 📦 Total Quantity
Calculates the total number of units sold across all transactions.

```DAX
Total Quantity =
SUM(Sales_data[Units Sold])
```

### 🧾 Transactions
Counts the total number of sales transactions recorded in the dataset.

```DAX
Transactions =
COUNTROWS(Sales_data)
```

### 📈 Average Price
Calculates the average selling price per unit.

```DAX
Average Price =
AVERAGE(Sales_data[Price Per Unit])
```

## 📅 Date Dimension

A dedicated Date Dimension was created using DAX to enable time intelligence analysis, drill-down reporting, and trend-based insights.

| Column | DAX Formula |
|---------|------------|
| Year | `YEAR([Date])` |
| MonthNo | `MONTH([Date])` |
| Month | `FORMAT([Date], "MMMM")` |
| QuarterNo | `INT(([MonthNo] + 2) / 3)` |
| Quarter | `"Qtr " & [QuarterNo]` |
| Day | `DAY([Date])` |

### 🎯 Business Purpose

- 📈 Year-over-Year (YoY) Analysis
- 📅 Month-wise Trend Tracking
- 📊 Quarterly Performance Analysis
- 🔍 Time-based Drill-down Reporting
- 🌟 Seasonal Pattern Identification
- 📉 Sales Trend Visualization

### 💡 Key Benefits

- Enables hierarchical date navigation
- Improves dashboard usability and reporting efficiency
- Facilitates advanced time intelligence calculations
